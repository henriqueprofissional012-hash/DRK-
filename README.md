# DRK Bombas e Filtros

Site institucional/comercial estático para a DRK Bombas e Filtros.

## Publicação na Vercel

Este projeto não precisa de build.

- Framework Preset: `Other`
- Build Command: deixar vazio
- Output Directory: deixar vazio
- Install Command: deixar vazio

## Arquivos principais

- `index.html`: estrutura do site e SEO básico.
- `styles.css`: identidade visual, responsividade e animações.
- `script.js`: menu mobile e dados de produtos/assistências/regiões.
- `assets/images`: imagens do projeto.
